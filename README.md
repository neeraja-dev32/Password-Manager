 🔐 Password Manager (Python OOP Project)

## 📌 Project Overview
This is a command-line based Password Manager application developed using Python.

The application allows users to store, manage, and retrieve login credentials such as site name, username, and password. It is built using Object-Oriented Programming (OOP) and includes file handling to store data permanently.

---

## 🚀 Features
- Add new passwords (site, username, password)
- View all saved credentials
- Delete stored passwords
- Data persistence using file handling
- Menu-driven user interface
- Case-insensitive search functionality
- Exit option to safely close the application

---

## 🧠 Concepts Used
- Python OOP (Classes, Objects, Methods)
- File Handling (read, write, overwrite)
- Lists & Data Structures

## ▶️ How to Run

1. Download or clone this repository  
2. Open the project folder  
3. Make sure Python is installed on your system  
4. Run the program using the command:  

   python password_manager.py

---

## 📂 File Structure

password_manager.py   # Main application file  
passwords.txt         # Stores user credentials  

---

## 💡 How It Works
- When the program starts, it loads existing data from the file.
- Users can interact with the system using menu options.
- Any changes (add/delete) are automatically saved.
- Data remains available even after restarting the program.
